## git sparse checkout

```sh
git clone [uri]
git config core.sparsecheckout true
echo [need path] > .git/info/sparse-checkout
git read-tree -m -u HEAD

cat .git/info/sparse-checkout

git config -l
```

ディレクトリの最期に / が必要らしい
