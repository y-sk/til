## How to install terraform

```sh
sudo yum install wget unzip

wget https://releases.hashicorp.com/terraform/0.12.7/terraform_0.12.7_linux_amd64.zip

sudo unzip ./terraform_0.12.7_linux_amd64.zip -d /usr/local/bin/

terraform -v
```
